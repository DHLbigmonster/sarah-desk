# Sarah 视频演示脚本 & 宣传文案

---

## 一、视频演示稿

> 时长建议：3-5 分钟。节奏要快，每段控制在 30-40 秒。

---

### 开场（10 秒）

**画面**：Sarah 菜单栏图标 + 浮动 HUD 同屏

> "这是 Sarah，一个住在菜单栏里的语音助手。按一下键开始说话，再按一下结束。不是 Siri 那种只会设闹钟的——它能帮你打字、帮你搜东西、帮你操作电脑。"

---

### 第一段：听写模式（40 秒）

**画面**：在 VS Code 或微信里按一下热键开始说话，文字直接出现

> "先看最基础的功能。我在任意 App 里，按一下 Right Ctrl，说一句话——"

*（演示：按一下热键开始说话，说完再按一下，文字出现在光标位置）*

> "再按一下，文字直接打到光标位置。不用切窗口，不用点麦克风，不用等加载。任何 App 都行——VS Code、微信、飞书、浏览器，都一样。"

*（切到设置界面）*

> "而且后面还有一层文字清洗——去口头禅、补标点、重组逻辑。你说的是口语，打出来的是书面语。"

**配图位置**：听写模式实际使用 GIF，展示"按一下→说话→再按一下→文字出现"的完整流程

---

### 第二段：Command 模式（50 秒）

**画面**：在一个网页或文档上，按触发键+Shift 说话

> "但听写只是基础。Command 模式才是真正有意思的部分。"

*（演示：在浏览器里按一下 Right Ctrl + Shift）*

> "我按一下触发键加 Shift，说一句'帮我搜一下 React 19 的新特性'——"

*（再按一下，Agent 覆盖层出现，显示执行过程）*

> "注意，Sarah 在弹出自己的界面之前，先截了一张当前屏幕的图。它看到的是我正在看的页面，不是它自己的界面。然后 Agent 就开始干活了——打开浏览器、搜索、读取内容、整理结果。"

*（结果展示）*

> "全程我没碰鼠标。而且 Agent 还能接入飞书——'帮我在飞书文档里新建一个会议纪要'，它直接调 API 去创建。'把这段话发到项目群里'，它就发了。"

**配图位置**：Command 模式完整流程 GIF——截屏→Agent 执行→结果展示

---

### 第三段：快问模式（30 秒）

**画面**：按触发键+Space，浮窗回答问题

> "第三个模式最轻量。按一下触发键加空格，问一个问题，AI 直接在浮窗里回答你。不用打开浏览器，不用切到 ChatGPT，问完继续干活。"

*（演示：按一下 Right Ctrl + Space，说"明天北京天气怎么样"，浮窗显示回答）*

**配图位置**：快问模式浮窗截图

---

### 第四段：零配置（20 秒）

**画面**：首次启动界面

> "最重要的一点——零配置就能用。下载下来，打开，授权麦克风，就能说话了。没有注册、没有 API Key、没有中国手机号。语音识别用的是 macOS 自带的 Apple Speech，离线也能用。想用更好的识别精度？去设置里配个火山引擎的 Key 就行。"

**配图位置**：首次启动 + 权限申请截图

---

### 第五段：技术亮点快速过（30 秒）

**画面**：五个窗口拆解图 / 调试控制台截图

> "技术上有几个点值得一提。Sarah 有五个独立窗口——录音、浮动 HUD、Agent 覆盖层、迷你设置、调试控制台，各司其职。凭证全部加密存储。所有配置可视化操作，不用碰命令行。开源，MIT 协议，代码全公开。"

**配图位置**：五个窗口拆解示意图 / ClawDesk 调试控制台截图

---

### 收尾（20 秒）

**画面**：GitHub 页面 + 应用图标

> "Sarah 是我用 Claude Code 一天做完的。对，一天。从架构到代码到 UI 到打包发布，全程 AI 开发。现在开源了，GitHub 链接在简介里。目前只支持 macOS，Windows 版在做。如果你也想要一个真正能用的语音助手，去试试。"

## 二、宣传文案

---

**Sarah —— The Siri you actually wanted.**

按一下键说话，再按一下结束。打字、搜东西、操作飞书，全靠一张嘴。

### 版本 B：朋友圈/即刻版（150 字以内）

做了个 macOS 语音助手叫 Sarah。三种模式：按一下说话→直接打字；加 Shift→AI 帮你干活；加空格→快问快答。全局热键，任何 App 都能用。零配置，下载即用。全程用 Claude Code 一天做完，已开源。GitHub: DHLbigmonster/sarah-desk

### 版本 C：飞书群详细介绍版（300 字）

> 推荐发到技术群或产品群

**Sarah：一个住在菜单栏的 AI 语音助手**

按一下 Right Ctrl 说话，再按一下文字直接出现在光标位置。任何 App 都行。

三种模式：
- **听写**：按一下说话→文字打到屏幕，带智能清洗（去口头禅、补标点）
- **命令**：加 Shift→AI Agent 帮你搜东西、操作飞书、打开网页
- **快问**：加空格→浮窗快速问答，不用切窗口

亮点：
- 零配置，下载就能用（Apple Speech 免费离线识别）
- 热键可自定义（Right Ctrl / CapsLock / F1-F12）
- 上下文感知——Agent 看到的是你正在看的屏幕，不是它自己
- 全部配置可视化操作，不用碰命令行

macOS M 芯片，MIT 开源：github.com/DHLbigmonster/sarah-desk

### 版本 D：技术社区/即刻长文版

> 适合技术社区发布

**用 Claude Code 一天做了一个 macOS 语音助手，开源了**

事情是这样的：我一直觉得 Siri 不够用。不是不够聪明，是根本没法用——你得先唤醒它，然后它还不一定理解你，理解了也不一定能执行。

所以我做了一个叫 **Sarah** 的工具。核心理念很简单：按一下键，说话，再按一下。就完事了。

三种模式：
1. **听写**：你说的话直接打到光标位置，任何 App 都行。后面还有一层文字清洗——去口头禅、补标点、重组逻辑。
2. **命令**：加个 Shift，AI Agent 帮你干活——搜索、操作飞书、打开网页，全程不碰鼠标。
3. **快问**：加个空格，浮窗快速问答，问完继续工作。

技术上有意思的点：
- 五个独立 Electron 窗口各司其职
- 上下文截图感知——Agent 在你按热键的那一刻就截了屏，看到的是你正在看的内容
- 凭证加密存储，所有配置可视化操作
- 零配置可用——macOS 自带 Apple Speech，离线也能用

全程用 Claude Code 开发。不是"AI 辅助写几个函数"，是把 Claude Code 当主力 IDE 从头到尾做完的。一天。

开源，MIT 协议：github.com/DHLbigmonster/sarah-desk
